sch
===

A small script used to generate an iCal .ics calendar file from McGill course CRNs.

Usage
-----

1. Shell  
```
./sch CRN1 CRN2 ... CRN2
```
2. wget  
```
wget "http://froyomuffin.com/cgi-bin/sch?CRN1+CRN2+...+CRNn" cal.ics
```  
3. Host it yourself!
